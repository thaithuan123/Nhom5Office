PHẦN 1: KIẾN TRÚC & CÔNG NGHỆ (45 giây)
---
Frontend chúng ta xây dựng trên nền tảng hiện đại:

📌 Framework: React 19 - thư viện JavaScript phổ biến nhất cho UI
📌 Build Tool: Vite - công cụ build nhanh, hot reload tức thời
📌 Routing: React Router DOM v6 - điều hướng trang SPA
📌 HTTP Client: Axios - gọi API backend một cách dễ dàng
📌 Styling: CSS Modules + Custom CSS - responsive design
📌 State Management: React Context API - quản lý state toàn cục
📌 Storage: LocalStorage - lưu trữ dữ liệu client-side

Ưu điểm của kiến trúc này:
✓ Performance cao với Vite's fast refresh
✓ Component-based architecture dễ maintain
✓ Responsive design cho mobile và desktop
✓ SEO-friendly với React Helmet (nếu cần)

PHẦN 2: CẤU TRÚC COMPONENTS & PAGES (1 phút)
---
Frontend được tổ chức theo cấu trúc component-based:

【COMPONENTS】(Reusable UI Elements)
- Header: Navigation bar với logo, menu, giỏ hàng, auth buttons
- ProductCard: Card hiển thị sản phẩm (hình, tên, giá, rating)
- ProductList: Grid layout cho danh sách sản phẩm
- Footer: Thông tin liên hệ, copyright
- AdminRoute: Bảo vệ routes admin (chỉ admin mới vào)
- ProtectedRoute: Bảo vệ routes user (phải login mới xem)

【PAGES】(Main Views - 12 pages)

🔹 PUBLIC PAGES (Khách hàng)
- Home: Trang chủ với hero banner, featured products
- Products: Trang shop với filter, search, sort
- ProductDetail: Chi tiết sản phẩm với specs, reviews
- Cart: Giỏ hàng với checkout form
- Auth: Đăng nhập/đăng ký cho khách hàng

🔹 ADMIN PAGES (Quản trị)
- AdminDashboard: Dashboard tổng quan (stats, quick actions)
- AdminProducts: CRUD sản phẩm (thêm/sửa/xóa điện thoại, quản lý tồn kho, upload ảnh và filter theo brand)
- AdminOrders: Quản lý đơn hàng (xem, update status)
- Users: Quản lý nhân viên/user (thêm/sửa/xóa user, phân quyền admin/user, tìm kiếm và kích hoạt/deactivate)

🔹 UTILITY PAGES
- NotFound: 404 page khi route không tồn tại

PHẦN 3: STATE MANAGEMENT - 3 CONTEXTS CHÍNH (45 giây)
---
Chúng ta sử dụng React Context để quản lý state toàn cục:

【1】AuthContext - Quản lý authentication khách hàng
- registeredUsers: Danh sách users đã đăng ký (localStorage)
- currentUser: User đang login
- Demo user: user@phonehub.com / user123
- Functions: login, logout, register

【2】AdminContext - Quản lý admin authentication
- adminUser: Admin đang login
- isAdmin: Boolean check quyền admin
- Demo admin: admin@phonehub.com / admin123
- Functions: loginAsAdmin, logoutAdmin

【3】CartContext - Quản lý giỏ hàng
- cartItems: Array sản phẩm trong giỏ
- Persistence: Lưu localStorage tự động
- Functions:
  - addToCart(): Thêm sản phẩm (smart merge nếu trùng)
  - removeFromCart(): Xóa sản phẩm
  - updateQuantity(): Thay đổi số lượng
  - clearCart(): Xóa toàn bộ giỏ

Logic thông minh:
- Khi add sản phẩm đã có → qty += 1 (không duplicate)
- Khi update qty ≤ 0 → tự động remove
- Tự động sync với localStorage

PHẦN 4: LOGIC XỬLÝ - API INTEGRATION & ROUTING (1 phút)
---
Frontend kết nối với backend qua REST API:

【API CONFIGURATION】
- Development: http://localhost:4000
- Production: https://backend-webbandienthoai.onrender.com
- Axios instance với base URL tự động

【ROUTING SYSTEM】
React Router với nested routes:
- Public routes: /, /products, /product/:id, /cart, /auth
- Admin routes: /admin, /admin/products, /admin/orders, /admin/users
- Protected routes: Sử dụng ProtectedRoute component
- Admin routes: Sử dụng AdminRoute component

【DATA FETCHING】
- useEffect hooks để fetch data khi component mount
- Loading states và error handling
- CRUD operations: GET, POST, PUT, DELETE cho tất cả entities

【FORM HANDLING】
- Controlled components với useState
- Validation client-side (required fields, email format)
- Real-time feedback cho user

【SEARCH & FILTER】
- Search by name/brand
- Filter by brand (Apple, Samsung, Xiaomi, OPPO)
- Sort by: newest, price low-high, price high-low, rating
